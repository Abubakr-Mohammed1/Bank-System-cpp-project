# 🏦 Bank System Extension (C++)

## 📖 Overview
This project is an extended version of the basic Bank System, developed using C++.  
It introduces transaction features and improves the overall structure of the system.

The system allows managing client accounts and performing banking operations through a console interface.

All data is stored in a text file to ensure persistence between runs.

---

## ⚡ Features

### 👤 Client Management
- 📋 Show all clients  
- ➕ Add new client  
- ❌ Delete client  
- ✏️ Update client information  
- 🔍 Find client  

### 💰 Transactions
- 💵 Deposit money  
- 💸 Withdraw money  
- 📊 View total balances  

---

## 💾 Data Storage
- Data is stored in: Clients.txt
- Each client contains:
  - Account Number  
  - PIN Code  
  - Name  
  - Phone Number  
  - Account Balance  

- Data is updated after every operation (Add / Delete / Update / Transactions)

---

## 🛠️ Code Structure
The system is designed using:

- Structs → to represent client data  
- Enums → to control menu navigation  
- Functions → to divide the program into small, manageable parts  

The program follows a structured approach using:
- Divide and Conquer  
- Clean and readable logic  

---

## ▶️ How It Works
1. The program starts with the Main Menu  
2. The user selects an operation  
3. The system processes the request  
4. Data is updated and saved to the file  

---

## 🧠 What This Version Adds
Compared to the basic version, this extension introduces:

- Transaction system (Deposit / Withdraw)  
- Balance validation  
- Total balance calculation  
- Better code organization
