# 🏦 Bank1 (Bank System - Version 1)

## 📖 Overview
This is a console-based Bank Management System developed using C++.  
The system allows managing client accounts and performing basic banking operations.

All data is stored in a text file to keep records saved between program runs.

---

## ⚡ Features
- 📋 List All Clients  
- ➕ Add New Client  
- ❌ Delete Client  
- ✏️ Update Client Information  
- 🔍 Find Client  
- 🚪 Exit the System  

---

## 💾 Data Storage
- Client data is stored in:  Clients.txt

- - Each client has:
  - Account Number  
  - PIN Code  
  - Name  
  - Phone Number  
  - Account Balance  

- Data is saved and updated after each operation.

---

## 🛠️ Code Structure
The system is built using:
- Structs to represent client data  
- Enums for menu options  
- Functions to organize and control the program logic  

The program loads data from the file, processes it, and saves changes when needed.

---

## ▶️ How It Works
1. The program starts with a main menu  
2. The user selects an option  
3. The system performs the operation  
4. Changes are saved to the file
